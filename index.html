Morse Code Transcriber

The length of an acceptable dot is 200 ms and the length of a dash is 600 ms.

This uses ITU Morse code. Use any key to generate a tone. The length of the tone decides if it is a dot or a dash. Note that the "invitation to transmit" code is exactly the same as the code for K. Adjust the slider to adjust the length of an acceptable Dash. According to the ITU (in terms of length), a dash is equal to three dots, the blank between characters is equal to three dots, and the space between two words is equal to seven dots.
